About

  • Configure a Emacs editor for latex compile in Windows environment.
  • An easy installation of Emacs as latex editor for newbie.

Change Log

  • Add: shortcut to desktop.
  • Add: register file for texfiles(*.tex, *.bib, *.sty, *.cls).
  • Add: tiny tools: 1) convert pdf to eps; 2) merge bibtex
  • Fix: Emacs.bat open file error, when filename has space.

  • Aspell 0.50.3 (4 Spelling Check)
  • backup-each-save 2011-08-03 (4 Backup Each Edition)

Install

UNZIP:

  • into %TEXLIVE_HOME%, exists %TEXLIVE_HOME%/install-tl*/install-t1-advanced.bat
  • into %EMACS_HOME%, exists %EMACS_HOME%/bin
  • into %AUCTEX_HOME%, exists %AUCTEX_HOME%/site-lisp/auctex
  • into %EMACS4LS_HOME%, exists %EMACS4LS_HOME%/bin ### COPY:
  • from %AUCTEX_HOME%/* to %EMACS_HOME%/ --> (Install Auctext into Emacs)
  • from %EMACS4LS_HOME%/* to %EMACS_HOME%/ --> (Install Emacs4LS into Emacs)
  • from %SumatraPDF*.exe to %EMACS_HOME%/bin/SumatraPDF.exe --> (Update SumatraPDF into Emacs) ### RUN:
  • run %TEXLIVE_HOME%/install-tl-*/install-tl-advanced.bat --> (Install Texlive 2012)
  • run %EMACS_HOME%/Emacs_Register.bat --> (Register .emacs plugins) ### PATH:
  • add %EMACS_HOME%\bin\ into %PATH% of OS Environment
  • add %EMACS_HOME%\aspell\bin\ into %PATH% of OS Environment ## Tiny Tools

Emacs_MGBib.bat

  • @usage: Emacs_MGBib.bat file1.bib file2.bib file3.bib ...
  • @description: merge bible files ### Emacs_PDF2PS.bat
  • @usage: Emacs_PDF2PS.bat file1.pdf file2.pdf file3.pdf ...

  • @description: convert pdf to eps (figures)
