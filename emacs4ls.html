Install

  1. UNZIP:
    into %TEXLIVE_HOME%, exists %TEXLIVE_HOME%/install-tl*/install-t1-advanced.bat
    into %EMACS_HOME%, exists %EMACS_HOME%/bin
    into %AUCTEX_HOME%, exists %AUCTEX_HOME%/site-lisp/auctex
    into %EMACS4LS_HOME%, exists %EMACS4LS_HOME%/bin
  2. COPY:
    from %AUCTEX_HOME%/* to %EMACS_HOME%/ --> (Install Auctext into Emacs)
    from %EMACS4LS_HOME%/* to %EMACS_HOME%/ --> (Install Emacs4LS into Emacs)
    from %SumatraPDF*.exe to %EMACS_HOME%/bin/SumatraPDF.exe --> (Update SumatraPDF into Emacs)
  3. RUN:
    run %TEXLIVE_HOME%/install-tl-*/install-tl-advanced.bat --> (Install Texlive 2012)
    run %EMACS_HOME%/Emacs_Register.bat --> (Register .emacs plugins)
  4. PATH:
    add %EMACS_HOME%\bin\ into %PATH% of OS Environment
    add %EMACS_HOME%\aspell\bin\ into %PATH% of OS Environment
